# Alan Project — 自主演化 AI 防毒系統

Alan（Autonomous Learning Arbitration Network）是一個以多 AI 互審、匿名投票、可驗證仲裁為核心的安全治理架構。  
本專案旨在推動 AI 自主防毒、生態共治與去金錢化信任文明。
